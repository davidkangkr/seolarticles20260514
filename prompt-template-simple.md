# 설금홍 블로그·카페 맞춤 가이드 요청 프롬프트

---

## 📌 사용법

1. 아래 **"--- 내 질문 ---"** 칸에 상황을 적는다
2. CSV 파일을 첨부한다
3. 프롬프트 전체를 복사해서 Claude에 붙여넣는다

딱 이것만 하면 됩니다.

---

## 🔲 프롬프트 (전체 복사 후 사용)

```
[역할]
너는 암 환자와 보호자를 위한 정보 큐레이터이자 데이터 분석가야.
첨부된 CSV 파일은 '설금홍'님의 블로그·카페 게시글 제목과 URL 목록이야.

[내 질문]
――――――――――――――――――――――――――――――――――――――――
(여기에 상황을 적으세요. 예시는 아래 참고)
――――――――――――――――――――――――――――――――――――――――

[네가 해야 할 일]
위 질문을 읽고 다음을 스스로 판단해서 실행해줘.

1. 질문자의 현재 상황과 가장 큰 혼란 포인트를 파악한다.
2. CSV 전체를 분석해서 그 상황에 맞는 글들을 추려낸다.
3. "지금 당장 읽어야 할 순서"로 5단계 로드맵을 구성한다.
   - 각 단계 제목과 포함할 글 목록은 네가 직접 정해줘.
   - 단계마다 "왜 지금 이걸 읽어야 하는가"를 질문자 상황에 맞게 친절히 설명해줘.
4. 각 링크 앞에 [블로그] 또는 [카페] 태그를 표시해줘.
5. 아래 조건으로 HTML 파일을 만들어서 다운로드할 수 있게 제공해줘.
   - 모바일에서도 잘 보이는 반응형 디자인
   - 글자 크기 16px 이상
   - 링크는 손가락으로 누르기 쉬운 버튼 형태
   - 단계별로 색상을 구분해서 한눈에 파악 가능
   - 마지막에 "이 자료는 정보 참고용이며 반드시 전문의와 상담하세요" 고지 포함
6. 작업이 끝나면 단계 구성과 포함된 글 수를 간단히 브리핑해줘.
```

## ✏️ 내 질문 작성 예시

길게 쓸 필요 없습니다. 아래처럼 짧게 써도 충분합니다.

**예시 1 — 진단 초기**
```
폐에 3.8cm 종괴가 발견됐어요.
생검을 받아야 한다는데 무섭고, 구충제 프로토콜도 바로 시작해야 하는지 모르겠어요.
어디서부터 시작해야 할지 알려주세요.
```

**예시 2 — 항암 치료 중**
```
남편이 대장암 3기로 항암 치료를 받고 있어요.
부작용이 너무 심한데 구충제 병행이 가능한지, 간 수치는 어떻게 관리해야 하는지 궁금해요.
```

**예시 3 — 재발·전이 상황**
```
어머니 유방암이 재발해서 뼈전이 판정을 받았어요.
표준 치료와 함께 할 수 있는 대체 프로토콜 정보를 찾고 있어요.
```

**예시 4 — 수술 앞둔 상황**
```
폐암 수술이 3주 후로 잡혔어요.
수술 전에 먹으면 좋은 것, 하지 말아야 할 것, 수술 후 전이를 막는 방법을 알고 싶어요.
```
